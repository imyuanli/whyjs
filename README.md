# whyjs


## Install

```bash
$ npm i --save whyjs
```
```bash
$ yarn add whyjs
```

## Using

```javascript
import y from 'whyjs'

y.isString('') // true
```
or
```javascript
import { isString } from 'whyjs'

isString(1) // false
```
